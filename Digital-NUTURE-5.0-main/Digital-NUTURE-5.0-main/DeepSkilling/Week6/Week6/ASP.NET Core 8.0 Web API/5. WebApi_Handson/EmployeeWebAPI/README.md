# Web API Handson 5

## Objective
Implement JWT Authentication and Role-Based Authorization.

## Technologies Used
- ASP.NET Core 8.0
- JWT Authentication
- Authorization
- C#

## Features
- Generate JWT Token
- Token Validation
- Role-Based Authorization
- Protected APIs
- Authentication Middleware

## Endpoints

GET /Auth

GET /Employee

## Output

- JWT Token generated successfully.
- Unauthorized users receive HTTP 401.
- Authorized users can access secured APIs.
- Role-based authorization implemented.
