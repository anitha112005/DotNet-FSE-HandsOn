# Week 6 – ASP.NET Core 8.0 Web API

## Overview

This repository contains the hands-on exercises completed as part of the **Cognizant Deep Skilling Program – Week 6**. The exercises focus on building RESTful Web APIs using **ASP.NET Core 8.0**, implementing authentication and authorization using **JWT**, custom filters, exception handling, CRUD operations, and integrating **Apache Kafka** with C# applications.

---

## Technologies Used

- ASP.NET Core 8.0 Web API
- C#
- .NET 8 SDK
- RESTful APIs
- Swagger (OpenAPI)
- JWT Authentication
- Custom Authorization Filters
- Exception Filters
- Apache Kafka
- Confluent.Kafka NuGet Package
