# Web API Handson 3

## Objective
Implement Custom Filters and Exception Handling in ASP.NET Core Web API.

## Technologies Used
- ASP.NET Core 8.0
- C#
- Custom Authorization Filter
- Exception Filter

## Features
- Custom Authorization Filter
- Exception Filter
- Exception Logging
- Global Error Handling

## Output
- Unauthorized requests are blocked.
- Exceptions are logged successfully.
- API returns custom error messages.
