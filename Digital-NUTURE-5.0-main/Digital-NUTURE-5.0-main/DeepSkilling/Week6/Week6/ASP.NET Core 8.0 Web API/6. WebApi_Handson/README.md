# Kafka Integration with C#

## Objective
Develop a Kafka-based Chat Application using C#.

## Technologies Used
- C#
- .NET Console Application
- Apache Kafka
- Confluent.Kafka NuGet Package

## Projects

### Kafka Producer
Publishes chat messages to a Kafka topic.

### Kafka Consumer
Consumes chat messages from the Kafka topic.

## Features
- Kafka Producer
- Kafka Consumer
- Publish Messages
- Consume Messages
- Console-based Chat Application

chat-topic
